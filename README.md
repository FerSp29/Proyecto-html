# Portafolio Personal y Galería de Mascotas

Este proyecto consiste en el desarrollo de un sitio web estático compuesto por dos páginas: un **Portafolio Personal** y una **Galería de Mascotas**. El objetivo principal fue aplicar los conocimientos de maquetación semántica con **HTML5** y diseño moderno con **CSS3** siguiendo los principios de Experiencia de Usuario (UX).

## Descripción del Proyecto

El sitio web está diseñado con una estética "clean" y profesional utilizando una paleta de colores oscura para el encabezado y tonos claros para el cuerpo permitiendo que el contenido resalte mediante las sombras suaves y bordes redondeados.

## Características Principales:
* **Diseño Responsivo:** Estructura adaptada para una visualización clara.
* **Navegación Fluida:** Implementación de *Smooth Scroll* para que los desplazamientos internos sean mas elegantes y fluidos.
* **Sección Hero:** Presentación de alto impacto con nombre y el rol profesional.
* **Semántica HTML5:** Uso correcto de etiquetas como <header>, <nav>, <main>, <section> y <article>.

## Tecnologías Utilizadas

* **HTML5:** Estructura y contenido semántico.
* **CSS3:** Estilos personalizados, Flexbox, CSS Grid y efectoos de transición.
* **Google Fonts:** Tipografía moderna para mejorar la legibilidad.
